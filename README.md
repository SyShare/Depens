# Depens
Android Gradle Plugin、管理项目依赖

使用方式 ：[参考链接 https://blog.csdn.net/lisdye2/article/details/52292896]

 use Maven ：
              <dependency>
                <groupId>com.depend</groupId>
                <artifactId>depes</artifactId>
                <version>1.0</version>
                <type>pom</type>
              </dependency>

  use gradle : compile 'com.depend:depes:1.0'
